Loop feedback plugin gives premium users feedback tool in order to leave feedback for their web applications, and websites. 

== Description ==

This loop feedback tool is used to send feedback for users, enterprises website, and web application. The users, enterprises made their campground on loopinput.com can take their own code snippet which can be embeded in their website. 
Once they install this plugin and copy snippet into the form, the feedback tool is used to send feedback images, description into Loop's campground.
Therefore, clients can see their own websites, web apps bug, layout, imgages and font etc.

A few notes about the sections above:

*   It is only used for the premium users in loop
*   Anyone can use this plugin but the code snippet only be published from Loop.
*   The feedback image, description or email are sent to Loop campground comments.

== Installation ==

Upload the Loop Feedback plugin to your website, activate it, and then enter code snippet provided from loopinput.com. 

e.g.

1. Install Loop Feedback plugin on your wordpress plugin directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Place code snippet provided from https://loopinput.com on Screenshot Plugin Code.


== Frequently Asked Questions ==

= What is Loop feedback? =

Loop feedback is the feedback tool which can be embeded in user's website and let people leave errors, bugs about images and font.

= What about Screenshot plugin Code? =

This code snippet is provided from loopinput.com, which is only given to premium users. They copy this codesnippet into Screenshot plugin Code form to show Give Feedback button on the bottom-right of their websites.

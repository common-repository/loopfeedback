The Loop feedback plugin gives Loop premium users the visual feedback tool in order to collect feedback for their web applications and websites. 

== Description ==

Customer feedback is the fuel that improves your product and grows your revenue. Loop develops visual customer feedback management software, including a screenshot plugin that integrates directly into your website and an embeddable forum to vote on submitted feedback, to help you drive strategic product decisions and catch bugs.

A few notes about the sections above:

*   The feedback plugin can only be a accessed by Loop premium users
*   Anyone can use this plugin but the code snippet can only come from Loop
*   The feedback image, description and email are sent to your Loop feedback portal

== Installation ==

Create a feedback portal on app.loopinput.com then enter the screenshot tool code snippet provided within your feedback portal

e.g.

1. Install Loop Feedback plugin on your Wordpress plugin directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Place code snippet provided from https://loopinput.com on Screenshot Plugin Code.


== Frequently Asked Questions ==

= What is the Loop feedback tool? =

The Loop feedback tool is an in-app plug in that is perfect for gathering responses on specific parts of your app. Take screenshots and annotate on specific parts of your app to get detailed feedback.

= What about screenshot plugin code? =

This code snippet is provided from app.loopinput.com and only available to premium users. Copy the code snippet into Screenshot plugin Code form to show Give Feedback button on the bottom-right of your website.
